# Data Safety - Respostas Sugeridas

Estas respostas devem ser revisadas na Play Console conforme as perguntas exibidas na conta.

## Coleta de dados

Nesta versao, o app nao envia dados para servidor proprio e nao cria conta de usuario. Pagamentos e assinatura sao processados pela Google Play. A voz em alta qualidade, texto para voz e reconhecimento de voz podem depender dos servicos configurados no Android do aparelho.

## Dados acessados no dispositivo

- Textos visiveis na tela: usados para leitura em voz alta por acessibilidade.
- Notificacoes: usadas para falar mensagens recebidas.
- Audio do microfone: usado somente quando o usuario toca em "Falar" para comando de voz.
- Camera: aberta por acao do usuario para fotografar texto fisico.
- Assinatura: consultada pela Google Play Billing para liberar ou restaurar o acesso.

## Compartilhamento

O app nao vende dados pessoais e nao compartilha conteudo lido com terceiros.

## Finalidade

- Acessibilidade
- Funcionalidade do app

## Criptografia em transito

Nao se aplica a esta versao para conteudo lido, pois nao ha envio para servidor proprio.

## Exclusao de dados

Como nao ha conta nem historico permanente nesta versao, nao existe dado de conta para excluir. O usuario pode desativar as permissoes do app e o servico de acessibilidade nas configuracoes do Android.
