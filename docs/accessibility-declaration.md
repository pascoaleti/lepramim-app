# Declaracao de Uso da AccessibilityService API

O LePraMim usa a `AccessibilityService API` para ler em voz alta textos visiveis na tela e notificacoes de mensagens.

## Finalidade

A permissao de acessibilidade existe para a funcao principal do app: ajudar pessoas que nao sabem ler, idosos e pessoas com dificuldade de leitura a ouvirem mensagens, telas de aplicativos e textos do celular.

## Como funciona

1. O usuario ou cuidador ativa o servico de acessibilidade nas configuracoes do Android.
2. O usuario abre um aplicativo, mensagem ou tela com texto.
3. O usuario toca no botao **OUVIR** do LePraMim.
4. O LePraMim identifica textos visiveis e fala o conteudo em voz alta.

## Uso de dados

O conteudo acessado pela API e usado para leitura em voz alta. O LePraMim nao usa servidor proprio para guardar mensagens ou historico de leitura, nao vende dados pessoais e nao usa o conteudo lido para publicidade.

## Limites de uso

O LePraMim nao usa o servico de acessibilidade para controlar o aparelho sem autorizacao, fazer compras, enviar mensagens, alterar configuracoes, coletar senhas ou executar acoes automaticas em nome do usuario.

O usuario pode desativar o servico a qualquer momento nas configuracoes de acessibilidade do Android.
